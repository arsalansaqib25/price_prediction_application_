# 🚗 Used Car Price Prediction Using Machine Learning

## 📌 Project Overview

The **Used Car Price Prediction** system is a Machine Learning-based web application that estimates the market price of used cars in Pakistan. The application helps buyers and sellers make informed decisions by predicting a fair selling price based on vehicle specifications.

The project was developed using Python, Machine Learning, and Streamlit, with a clean and interactive dashboard for prediction and market analysis.

---

## 🎯 Objectives

* Predict the market price of used cars.
* Compare multiple Machine Learning algorithms to identify the best-performing model.
* Provide users with an easy-to-use web interface.
* Visualize market trends through interactive dashboards.

## 🤖 Machine Learning Models

The project evaluates multiple regression algorithms, including:

* Linear Regression
* SVR
* Random Forest Regressor
* XGBoost
* CatBoost

The best-performing model is used for the final prediction system.

---

## 📈 Model Evaluation

The models are evaluated using:

* R² Score
* Mean Absolute Error (MAE)
* Root Mean Squared Error (RMSE)

## 🚀 Running the Project Locally

### 1. Clone the Repository

```bash
git clone https://github.com/Arsalansaqib25/Used-Car-Price-Prediction-.git
```

### 2. Move into the Project Folder

```bash
cd Used-Car-Price-Prediction
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Streamlit Application

```bash
streamlit run Home.py
```
